MIST: Medical Image Streaming Toolkit

A unified framework for intelligent, progressive, and resource-efficient medical image streaming.

Journal Paper (JDI 2024) Conference Paper (MIDL 2025) Patent GitHub

Overview

MIST introduces a format-agnostic, open-source framework for progressive streaming of medical imaging data. By transmitting only the optimal sub-resolution needed for AI inference or visualization, MIST enables dramatic reductions in bandwidth, storage, and compute requirements while preserving diagnostic fidelity.

  • Progressive multi-resolution streaming using HTJ2K
  • AI-aware stream optimization
  • Standardized Python interface (OpenJPHpy)
  • Compatibility with DICOM, NIfTI, and standard image formats
